# Курсовая работа: прогнозирование эффективности химических соединений<br>

Исследование данных о 1000 соединениях с показателями **IC50**, **CC50** и **SI** для подбора параметров разработки противогриппозных препаратов<br>

## Структура репозитория<br>

| Курсовая_работа_Файл_1_Eda_Урамаев.ipynb` | EDA, предобработка, формирование целевых признаков<br>
| Курсовая_работа_Файл_2_Регрессия_IC50_Урамаев.ipynb | Регрессия IC50<br>
| Курсовая_работа_Файл_3_Регрессия_CC50_Урамаев.ipynb | Регрессия CC50<br>
| Курсовая_работа_Файл_4_Регрессия_SI_Урамаев.ipynb | Регрессия SI<br>
| Курсовая_работа_Файл_5_Классификация_для_IC50_медианы_Урамаев.ipynb | IC50 > медианы<br>
| Курсовая_работа_Файл_6_Классификация_для_CC50_медианы_Урамаев.ipynb | CC50 > медианы<br>
| Курсовая_работа_Файл_7_Классификация_для_SI_медианы_Урамаев.ipynb | SI > медианы<br>
| Курсовая_работа_Файл_8_Классификация_для_SI___8_Урамаев.ipynb | SI > 8<br>

Аналитический отчёт - Аналитический_отчёт_Урамаев.PDF<br>
Сериализованный датасет после обработки - dataset_after_eda.pkl<br>

## Данные<br>

1. Исходный датасет: Данные_для_курсовои_Классическое_МО.xlsx<br>
2. После EDA сохраняется dataset_after_eda.pkl.<br>

## Запуск<br>

**В следующем порядке:** сначала файл 1, затем 2-8 в любом порядке.<br>

1. Положить xlsx в корень репозитория или указать путь в ноутбуке 1<br>
2. Выполнить Курсовая_работа_Файл_1_Eda_Урамаев.ipynb, появится dataset_after_eda.pkl<br>
3. Запускать остальные ноутбуки из той же папки<br>

При локальном запуске заменить на:<br>

df = pd.read_pickle("dataset_after_eda.pkl")<br>

1. Python 3.10<br>
2. pandas, numpy, matplotlib, seaborn, openpyxl<br>

## Модели<br>
В каждой задаче: сравнение baseline-моделей, использованы: линейные / логистическая регрессия, Random Forest, Gradient Boosting, CatBoost и подбор гиперпараметров через GridSearchCV.
